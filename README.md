Spring Boot Project Sub Module

    1 - pgvector-insert-retrieval : Inserts data into PgVector and retrieves it using embedding similarity search.

    2 - spring-vector-database : Implements semantic search using Spring AI with multiple vector databases (PgVector, Redis, etc.) to store embeddings and retrieve relevant data based on similarity.

    3 - spring-ai-metadata-filter : Applies metadata-based filtering during similarity search..

    4 - tool-calling-spring-ai : Demonstrates annotation-based Tool Calling and Programmatic Tool Calling using Spring AI.

    5 - image-audio-generation : This submodule demonstrates AI capabilities including Prompt-to-Image generation, Speech-to-Text transcription, and Text-to-Speech conversion

    6 - ollama-spring-boot : This submodule demonstrates integration with Ollama to run Large Language Models locally

    7 - spring-observability : This submodule demonstrate monitoring of AI Calls

    8 - SpringAI-ETL-PipeLine : This submodule demonstrates the ETL (Extract, Transform, Load) workflow by extracting data

    9 - OpenAi : This submodule demonstrates pattern of Spring AI  like - Chain Pattern WorkFlow
                                                                    - Parrallel WorkFlow
                                                                    - Routing WorkFlow
                                                                    - Orchestrator WorkFlow 
                                                                    - Evalutor-Optimizer Patter
