
Project Description 
           This repository contains two Spring Boot projects demonstrating vector insertion and retrieval using Spring AI and pgvector.
                - springai-pgvector-insert
                - springai-pgvector-retrival 


                springai-pgvector-insert : In this project, I insert vector embeddings into PostgreSQL using pgvector.

                - Tech Stack 
                    Dependency 
                        - Spring Web
                        - Spring-Ai-Advisors-Vector-Store
                        - Spring-Ai-Starter-Model-Openai
                        - Spring-Ai-Starter-Vector-Store-PGVector
                        - Postgresql

                    Docker Image 
                             : PostgreSQL with pgvector extension:
                             : pgvector/pgvector:pg16

                     Java Version - 17
                     Spring-AI Version - 2.0.0-M2
                     Spring Boot Version - 4.0.2        


                 springai-pgvector-retrieval  : In this project In this project, I retrieve records based on semantic similarity.

                  - Tech Stack 
                    Dependency 
                        - Spring Web
                        - Spring-Ai-Advisors-Vector-Store
                        - Spring-Ai-Starter-Model-Openai
                        - Spring-Ai-Starter-Vector-Store-PGVector
                        - Postgresql    

                    Java Version - 17
                     Spring-AI Version - 2.0.0-M2
                     Spring Boot Version - 4.0.2
