Project Description 
            This project is a Spring Boot application that integrates OpenAI services using Spring AI. It demonstrates multiple AI capabilities, including

            - Text-to-Image generation
            - Speech-to-Text conversion
            - Text-to-Speech generation


     Tech Stack : 
            Java Version 21 
            Spring AI Version 2.0.0-M2
            Spring Boot Version 4.0.2

            Dependencies     
               - Spring Web
               - OpenAi
               - Lombok
