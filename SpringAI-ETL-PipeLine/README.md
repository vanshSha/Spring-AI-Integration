Project Description 
        This Spring Boot project demonstrates a basic ETL (Extract, Transform, Load) pipeline.
        The application supports data extraction and processing from multiple sources and includes enrichment capabilities

           - ETL Enrichment
           - ETL for PDF files
           - ETL for JSON data
           - ETL for Markdown files

        Tech Stack 

                Java Version 17
                Spring Boot Version 4.0.2
                OpenAi 2.0.0-M2

           Dependency 
                Spring Web
                OpenAi
                PDF Document Reader
                Markdown Decument Reader
