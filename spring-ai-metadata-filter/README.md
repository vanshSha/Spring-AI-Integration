Project Description :
                    This project demonstrates the implementation of metadata filtering using a vector database.
                    It allows filtering search results based on metadata fields such as category, brand, or other attributes while performing semantic search.

    Tech Stack  : 
                Java Version 17 
    
    Dependencies : Spring-web
                 : OpenAi
                 : Lombok
                 : PgVectore-Store 
                
    Docker Image : pgvector
